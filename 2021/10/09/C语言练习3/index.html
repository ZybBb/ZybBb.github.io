C语言练习3

发表于 2021-10-09 更新于 2021-10-21

##while执行语句也是挺有趣的

1、这是3秒的倒计时

#include<stdio.h>
#include<stdlib.h>
#include<windows.h>
int main()
{
int a;
a=3;
while(a>0)
{
system(“cls”);
printf(“%d”,a);
a=a-1;
Sleep(1000);
}
system(“pause”);
return 0;
}

cls是每个数字闪过后清楚，Sleep是控制闪过的速度，可以去试试Sleep(50)

2、100以内的数之和

#include<stdio.h>
#include<stdlib.h>
int main()
{
int a,i;
a=0;
i=1;
while(i<=100)
{
a=a+i;
i=i+1;
}
printf(“%d\n”,a);
system(“pause”);
return 0;
}
